# ch1. 운영체제 소개

## 1.1 운영체제?
### 한줄 요약
- 컴퓨터 시스템은 크게 hw, sw, 시스템sw로 구성되며 대표적인 시스템 sw가 운영체제.
- 역할=자원관리+사용자지원 -> 시스템의 효율성 높임
- 커널(=일체형vs마이크로)모드 vs 사용자모드: 하드웨어 제어 여부, trigger=시스템호출

## 1.2 운영체제의 구성
### 한줄 요약
- 프로세스 관리자(생성,삭제,스케줄) + 메모리 관리자(요구처리,할당,회수,보호) + 파일 관리자(접근제한,메모리) + 장치 관리자(할당,작동,반환)

## 1.3 운영체제의 유형
### 한줄 요약
- 일괄처리(batch)os, 시분할os(=interactive), 실시간os(작업 우선순위), 분산os(분산시스템)

## 키워드 흐름
- 역할(2가지->모드), 구성요소(process,memory,file,device-관리자), 종류(batch,time-share,real-time,distribute)
